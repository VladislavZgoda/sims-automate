Активировать venv:
.\.venv\Scripts\activate

Запуск скрипта:
uv run .\main.py

Необходимо создать файл profiles.json с данными.
К примеру:
[
  {
    "file_name": "Тест",
    "serial_number": "12345678"
  }
]

При запуске Sims должен быть открыт с Управление группами в панели MDI.
